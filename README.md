# 📊 Trader Performance vs Market Sentiment

## 📌 Objective
Analyze how market sentiment (Fear/Greed) affects trader behavior and performance.

## 📂 Datasets
- Fear & Greed Index
- Historical Trader Data

## ⚙️ Methodology
- Data cleaning and merging
- Feature engineering (PnL, leverage, trades)
- Sentiment-based analysis

## 📊 Key Insights
- Higher profits during Extreme Greed
- Increased trading activity in Greed markets
- High leverage leads to higher risk

## 🚀 Strategy Recommendations
- Reduce leverage during Fear markets
- Trade actively but carefully during Greed
